Endfield Model Importer

Mod Installation:
1. Extract mod's archive
2. Put extracted folder into the Mods folder

Mods Folder Location:
1. Click 3-dots on [Start] button of XXMI Launcher
2. Click [Open Mods Folder]
Default path is `%APPDATA%/XXMI Launcher/EFMI/Mods`

Mod User Hotkeys:
[F10]: Reload EFMI and Save mods settings
[F11]: Toggle Mods display
[F12]: Toggle this User Guide
[Ctrl]+[Alt]+[F10]: Reset Mod Settings and Reload

Mod Developer Hotkeys:
[Ctrl]+[F10]: Toggle Perfomance Monitor
[Ctrl]+[F11]: Disable custom shaders while held
[Ctrl]+[F12]: Toggle Shader Hunting Mode Guide
